welcome to my student enquiry management system analysis project:

The use case is based on a hypothetical scenario whereby a university is facing operational challenges regarding managing student enquiries efficiently.
The problem that arises is that the students of the university struggle to receive assistance or information from the university. The university is seeking an analysis and proposed solution to this problem.

The powerpoint attached contains of a summarized analysis as a presentation.
It consists an introduction, problem statement, current state analysis, root cause analysis, business requirements, proposed solution, the implementation plan and an analysis summary.
