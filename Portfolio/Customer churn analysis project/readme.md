welcome to my customer churn analysis project using Kaggle data

This short use case analysis uses Kaggle customer data of Telco service provider.
This analysis contains a problem statement, the analysis objective and measurable success criteria, stakeholder requirements, data analysis using excel and powerbi, and recommended solutions. 

The output also includes a dashboard to visually portray the data. The dashboard elements include:
KPI card - total customers, total churn customers
Donut charts - distribution of churn based on customer contract type, distribution of churn based on customer payment method
Stacked column chart - churn based on customer demographic (senior citizen, has dependents, has a partner) that can drill up/down to compare the different factors. The total charges of customers based on their tenure. 
Clustered bar chart - churn rate of each customer group (new, developing, established, loyal)
Some measures created include the customer group and the churn rate.
